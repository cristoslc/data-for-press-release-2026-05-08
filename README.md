# Data for Press Release — May 8, 2026

This repository contains the raw data, analysis pipelines, and outputs supporting the May 8, 2026, No Flock for SoPo press release revealing nationwide police access to South Portland's Flock Safety surveillance cameras.

## What's here

- **`data/`** — FOAA-obtained records from SPPD: Flock audit logs, agency access lists, search query logs
- **`analysis/`** — Reproducible pipelines that transform raw records into press-release factoids
- **`outputs/`** — Verifiable artifacts: cross-reference of ICE-collaboration agencies, search-frequency summaries, geographic distribution

## Quick start

```
./run.sh          # Reproduce all outputs from raw data
```
